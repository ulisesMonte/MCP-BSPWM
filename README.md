# BSPWM

Mis dotfiles personales para **bspwm**.  
Setup minimalista, enfocado en velocidad y teclado.

---

## Qué incluye

- **bspwm** → window manager
- **sxhkd** → atajos de teclado
- **polybar** → barra
- **picom** → compositor
- **kitty** → terminal
- **neovim** → editor
- **GTK / Qt** → theming
- **scripts propios** para estados, red, VPN, etc.

---

## Estructura

```text
bspwm/        configuración principal del WM
sxhkd/        keybindings
polybar/      barra + scripts + fuentes
picom/        compositor
kitty/        terminal
nvim/         config de Neovim
gtk-3.0/      tema GTK
qt5ct/        tema Qt5
qt6ct/        tema Qt6
bin/          scripts que uso con bspwm/polybar
